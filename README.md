# 用户管理系统

## 基于 json-server 服务器，需先安装json-server服务器，再 npm run json:server 运行后台服务器，才能进行增删改查数据

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files

npm run lint
